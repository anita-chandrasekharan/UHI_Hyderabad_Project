# GeoAI-Based Urban Heat Island Analysis for Hyderabad

## Project Overview

This project analyzes Urban Heat Island (UHI) patterns in Hyderabad, India using multi-temporal satellite remote sensing datasets and GeoAI techniques.

The study integrates:
- Landsat thermal imagery
- Land Surface Temperature (LST)
- Vegetation and built-up indices
- Machine learning models
- Spatial hotspot analysis

## Objectives

- Map Urban Heat Island intensity
- Analyze relationships between urbanization and temperature
- Identify heat vulnerability zones
- Develop predictive GeoAI models for urban heat analysis

## Study Area

Hyderabad, Telangana, India

## Tools & Technologies

- Google Earth Engine
- Python
- QGIS
- Landsat 8/9
- Scikit-learn
- XGBoost

## Project Workflow

1. Satellite data acquisition
2. Image preprocessing
3. Land Surface Temperature estimation
4. NDVI/NDBI calculation
5. Hotspot analysis
6. Machine learning modeling
7. Visualization and mapping
